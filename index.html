Two-Deck DJ YouTube Player with Looping

Enter a YouTube link for each deck, then control each independently.
Keyboard Shortcuts:
• **Space**: Play/Pause (Shift for Deck 2)
• **C**: Set Cue (Shift for Deck 2)
• **V**: Jump to Cue (Shift for Deck 2)
• **O**: Set Loop Start (Shift for Deck 2)
• **P**: Set Loop End (Shift for Deck 2)
• **L**: Toggle Loop (Shift for Deck 2)
• **Arrow Up/Down**: Increase/Decrease speed (Shift for Deck 2)
